# Images Directory

## Background Image

Please copy your Peri Peri Grill restaurant photo to this directory and name it `background.png`.

The background image will be used on:
- The intro/welcome page (index.html)
- The menu selection page (menu.html)

**Current expected file:** `images/background.png`

If your image has a different name, either:
1. Rename it to `background.png`, OR
2. Update the CSS file (`styles/main.css`) to reference your image filename
